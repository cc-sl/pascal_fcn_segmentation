
## 使用步骤

1. **配置数据集路径**：修改 `config.py` 中的 `ROOT` 指向 VOCdevkit 解压后的路径。
2. **确保原始 VOC 2012 数据存在**，且 `ImageSets/Segmentation/train.txt`、`val.txt` 存在。
3. **安装依赖**：
   ```bash
   pip install torch torchvision pillow matplotlib numpy pandas tqdm
   ```
4. **单独训练一个模型**（例如 resnet18）：
   ```bash
   python train.py --backbone resnet18
   ```
5. **评估**：
   ```bash
   python evaluate.py --backbone resnet18
   ```
6. **可视化**：
   ```bash
   python visualize.py --backbone resnet18 --num_samples 3
   ```
7. **完整对比实验**（依次训练并评估两个模型，输出表格）：
   ```bash
   python compare_experiments.py
   ```

## 注意事项

- 训练和评估时会自动将类别映射到5类（背景、人、车、猫、狗），其他类别标记为忽略。
- 数据增强采用在线随机水平翻转，尺寸固定为256×256。
- 模型FCN-8s融合了layer2/3/4的特征，上采样采用转置卷积。
- 所有超参数在 `config.py` 中统一调整，保证对比实验公平。
