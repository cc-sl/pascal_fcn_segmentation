一、实验目的

l 深入理解语义分割的像素级分类本质，掌握FCN（全卷积网络）的核心思想、网络结构与工作原理，理解其作为语义分割开山之作的历史意义与局限性。

l 完整实现从数据预处理、模型训练、性能评估到推理可视化的全流程，掌握语义分割模型的工程化实现方法。

l 熟练计算并解读语义分割两大核心指标：mIoU、像素准确率，能基于指标多维度分析模型性能。

l 对比不同模型结构、训练策略对分割性能的影响，能对自己的分割结果进行简单的可视化和分析

二、实验内容

**1.基础任务**

使用FCN模型，在PASCAL VOC 2012语义分割数据集（5类子集）上完成多类别语义分割任务。

分割类别：背景、人、车、猫、狗共5类，覆盖VOC中最常见且标注质量最高的类别

**2.核心实验要求**

**（1）基础流程**

l 预训练FCN模型加载与训练

l 模型评估

l 推理可视化

**（2）对比实验**

l 编码器主干对比实验。固定所有训练超参数（学习率、batch size、epochs、数据增强、输入尺寸等），仅更换FCN的编码器主干网络。

模型1：FCN+ResNet18

模型2：FCN+ResNet34

对比指标：分割精度、训练效率、模型大小。

**3. 可选拓展任务**

尝试更换编码器，对比三者的速度。实现不同的上采样结构，对比不同上采样对分割精度的影响。将分类任务扩展到VOC完整的20类任务。
